# Gemini API Integration (Task B)

## 🚀 Overview
This project integrates with the Gemini 2.5 Flash API.

It exposes a simple FastAPI endpoint that:
- Accepts a prompt
- Sends it to Gemini API
- Returns generated response

---

## 🛠 Tech Stack
- Python
- FastAPI
- Requests
- Gemini API

---
